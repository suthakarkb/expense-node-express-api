# expense-nodejs-rest-api
**A node.js API server for MySQL**
- RESTful
- API OATH2 Authentication With Google

# API Inventory list
* POST: /user
* GET: /expenses
* GET: /expense/id
* POST: /expense
* PUT: /expense
* DELETE: /expense
* GET: /datatype/type

# Configure
default port = 3000;

# Compilation
npm install

# Start server
npm start

# Run tests
npm test
